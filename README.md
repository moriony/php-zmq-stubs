php-zmq-stubs
=============

ZeroMQ classes stubs for IDE autocomplete

Add in your ```composer.json``` the require entry for this library.
```json
{
    "require-dev": {
        "moriony/php-zmq-stubs": "dev-master"
    }
}
```
and run ```composer install --dev``` (or ```update --dev```) to download all files.
